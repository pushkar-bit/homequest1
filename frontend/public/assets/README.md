Place the Home background image here

- Add your `home-bg.jpg` file to this folder.
- The Home page expects `/assets/home-bg.jpg` to be present.
- Recommended size: at least 1920x1080 JPG/PNG.
- For better performance, optimize the image (webp or compressed jpg) before placing it here.

Usage:
- The Home page code uses this path: `/assets/home-bg.jpg`.
